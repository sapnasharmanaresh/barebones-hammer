Colophon

I've included a lot of abstrast classes that I regularly use for my daily web projects. Instead of including jQuery I'm using Zepto because I've realised most of the time I'm just adding toggling classes. Did I say it's a Hammer template as well. I've included some basic styling to allow you to extend it yourself.

Features

    * Uses 'normalize.css' by [Nicolas Gallagher](https://github.com/necolas) * Uses Zepto instead of jQuery * SMACSS folder layout * Added module abstracts * Px to em and px to rem mixin * Media object by [Nicole Sullivan](http://www.stubbornella.org/content/2010/06/25/the-media-object-saves-hundreds-of-lines-of-code/) * Flag object by [Harry Roberts](http://csswizardry.com/2013/05/the-flag-object/)
