Features

  • Uses 'normalize.css' by Nicolas Gallagher
  • Uses Zepto instead of jQuery
  • SMACSS folder layout
  • Added module abstracts
  • Px to em and px to rem mixin
  • Media object by Nicole Sullivan
  • Flag object by Harry Roberts

I've included a lot of abstrast classes that I regularly use for my daily web projects. Instead of including jQuery I'm using Zepto because I've realised most of the time I'm just adding toggling classes. Did I say it's a Hammer template as well. I've included some basic styling to allow you to extend it yourself.
