Barebones is a style guide that uses a SMACSS approach. I've included a lot of abstrast classes that I regularly use for my daily web projects. Instead of including jQuery I'm using Zepto because I've realised most of the time I'm just adding toggling classes. Did I say it's always a Hammer template as well.
